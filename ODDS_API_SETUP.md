# Настройка Odds API для получения предстоящих и лайв матчей

## Как букмекеры получают данные?

Букмекеры используют дорогие коммерческие сервисы (Oddin.GG, PandaScore) стоимостью десятки тысяч долларов в месяц. Для бесплатного решения мы используем:

1. **The Odds API** - агрегирует данные от букмекеров (бесплатный план: 500 запросов/месяц)
2. **OpenDota API** - для получения данных о матчах из лиг
3. **Альтернативные методы** - парсинг публичных источников

## Получение бесплатного API ключа

1. Перейдите на https://the-odds-api.com/
2. Зарегистрируйтесь (бесплатно)
3. Получите API ключ в разделе "API Keys"
4. Бесплатный план включает:
   - 500 запросов в месяц
   - Доступ к данным о предстоящих матчах
   - Доступ к лайв матчам
   - Данные от множества букмекеров

## Настройка в приложении

1. Откройте файл `lib/services/odds_api_service.dart`
2. Найдите строку:
   ```dart
   static const String apiKey = 'YOUR_API_KEY_HERE';
   ```
3. Замените `YOUR_API_KEY_HERE` на ваш API ключ:
   ```dart
   static const String apiKey = 'ваш_ключ_здесь';
   ```

## Что делать без API ключа?

Приложение будет работать и без ключа Odds API, используя альтернативные методы:
- Поиск предстоящих матчей в активных лигах OpenDota
- Анализ недавних матчей для определения лайв статуса

Однако для получения полных данных о предстоящих и лайв матчах рекомендуется получить бесплатный ключ.

## Альтернативные источники данных

Если Odds API недоступен, приложение использует:
- OpenDota API для получения матчей из активных лиг
- Анализ времени начала матчей для определения предстоящих
- Проверка длительности матчей для определения лайв статуса
